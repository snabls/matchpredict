**MATCH PREDICT APPLICATION**
(I know that the correct way should be prediction but in Italian social media slang the word "predict" stays for prediction :-) )

This application was born for a funny scope.
In 2024 I decided to create a game with some friends of mine to enjoy the football European Cup in a different way.
From this idea we created a tournament based on match predictions. Each of my friends started sending me via a WhatsApp group a prediction for each Euro match including:
- The prediction of the final score
- The First Scoring Country (FSC)

On the base of this predictions, a score was calculated.
+ 100 points for guessing the exact result
+ 70 points for a goal difference of 1
+ 50 points for a goal difference of 2
+ 25 points for a goal difference of 3
+ 20 points for guessing the match outcome (1X2)
+ 30 points for guessing the FSC

Initially, these scores were calculated by a GAS code behind a Google Sheets project but,
inspired from a project I worked on for my last high school year for a middle school secretary management system with a Vue.js front-end and a GAS back-end, I decided to move the game architecture.
